# 城市停车资源管理系统部署说明

## 1. 环境要求

- Node.js 22 及以上
- npm 10 及以上
- JDK 17
- MySQL 8

## 2. 数据库初始化

1. 创建数据库并执行 `sql/schema.sql`
2. 导入 `sql/data.sql`

## 3. 后端部署

```bash
cd backend
./mvnw spring-boot:run
```

说明：

- 当前代码为保证答辩稳定性，默认使用内置演示数据层直接启动。
- 若后续需要接入真实 MySQL，可在此基础上继续扩展 JPA 实体、Repository 和配置。

## 4. 前端部署

```bash
cd frontend
npm install
npm run dev
```

构建命令：

```bash
npm run build
```
