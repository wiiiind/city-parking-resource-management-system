# 城市停车资源管理系统部署说明

## 1. 环境要求

- Node.js 22 及以上
- npm 10 及以上
- JDK 17
- MySQL 8

## 2. 数据库初始化

推荐直接执行：

```bash
sudo mysql < sql/bootstrap.sql
```

执行完成后会自动：

1. 创建数据库 `city_parking_resource_management`
2. 创建数据库应用账号 `city_parking_app`
3. 完成授权
4. 建立全部业务表
5. 导入初始化演示数据

## 3. 后端部署

```bash
cd backend
./mvnw spring-boot:run
```

说明：

- 当前后端业务逻辑已直接接入 MySQL。
- 启动前请确保 MySQL 服务已经运行，且已执行 `sql/bootstrap.sql`。

## 4. 前端部署

```bash
cd frontend
npm install
npm run dev
```

构建命令：

```bash
npm run build
```

## 5. 演示入口

- 车主登录：`owner / 123456`
- 管理员后台：通过首页“进入后台管理”按钮进入
- 新车主可通过首页注册功能创建账号并登录
