# 城市停车资源管理系统测试用例

| 编号 | 测试模块 | 测试场景 | 预期结果 |
| --- | --- | --- | --- |
| TC-01 | 登录 | 使用 `admin / 123456` 登录 | 登录成功，显示管理员视角 |
| TC-02 | 登录 | 使用错误密码登录 | 返回账号或密码错误 |
| TC-03 | 停车场管理 | 新增停车场信息 | 列表新增停车场，日志同步记录 |
| TC-04 | 车辆管理 | 新增车主车辆档案 | 车辆列表增加新车辆 |
| TC-05 | 车辆入场 | 选择车辆和停车场执行入场 | 生成在场停车记录，停车场空闲车位减少 |
| TC-06 | 重复入场 | 对已在场车辆再次执行入场 | 返回“不能重复入场”提示 |
| TC-07 | 车辆出场 | 对在场车辆执行出场结算 | 更新停车记录、生成订单、空闲车位增加 |
| TC-08 | 看板 | 刷新综合看板 | 指标卡片和流量趋势正常展示 |
| TC-09 | 调度建议 | 查看调度建议列表 | 显示各停车场占用率和建议动作 |
| TC-10 | 系统公告 | 查看公告和日志 | 公告信息和最新操作日志正常显示 |

## 兼容与演示验证

- 浏览器：Chrome、Edge 最新版本
- 前端：`npm run dev` 启动正常，页面布局在桌面和移动宽度下可展示
- 后端：`./mvnw spring-boot:run` 启动后可访问 `/api/dashboard/overview`
- SQL：`schema.sql` 与 `data.sql` 可用于 MySQL 8 初始化
